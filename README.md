# jakobhelton.github.io

Personal website of **Jakob M. Helton** — Astrophysicist, Evolving Universe Postdoctoral Fellow at Penn State, visual artist, and science communicator.

## Quick Start

```bash
# 1. Replace your repo contents with this package

# 2. Add your images to assets/images/:
#    avatar.jpg         — Your headshot (used on biography page)
#    hero_1.jpg         — Hero slideshow image (your photography)
#    hero_2.jpg         — Hero slideshow image (your photography)
#    art-preview.jpg    — Preview for art section on landing page
#    art-painting.jpg   — Thumbnail for painting gallery
#    art-midwest.jpg    — Thumbnail for Midwest/Southwest book
#    art-analog.jpg     — Thumbnail for analog photography
#    art-digital-photo.jpg — Thumbnail for digital photography
#    art-digital.jpg    — Thumbnail for digital art
#    art-film.jpg       — Thumbnail for home movies

# 3. Place your CV at assets/cv/JMH_CV.pdf

# 4. Update Google Scholar URL in index.html (replace PLACEHOLDER)

# 5. Migrate blog and art content from your existing site:
pip install requests beautifulsoup4 lxml
python scripts/migrate_content.py

# 6. Add ADS API token as GitHub secret (ADS_API_TOKEN)
#    Get from: https://ui.adsabs.harvard.edu → Account → API Token

# 7. Enable GitHub Pages (Settings → Pages → main branch, root)

# 8. Trigger first data update (Actions → Daily Data Update → Run workflow)
```

## Key Design Choices

- **Deep space dark theme** (#08090c base) with jade green (#00a878) accent
- **LaTeX-inspired typography**: Playfair Display (headings) + Libre Franklin (body)
- **No hyphenation**: Words never break across lines (hyphens: none)
- **Justified text**: All body text is justified for a clean academic look
- **Animated star field**: Tiny twinkling stars + faint galaxy smudges on every page
- **Dark/light mode toggle**: Persisted per user in localStorage
- **Fully responsive**: Works on iPhone, tablet, desktop; hamburger nav on mobile
- **JWST images credited**: Every NASA image includes credit + clickable source link

## Presentations Page

The presentations page supports colloquia, public lectures, and conference talks. Each entry can include:

1. **Google Slides link** — Links directly to your presentation
2. **PDF download** — For offline viewing
3. **AI-narrated audio** — Provide a script per slide; audio is generated via TTS and synchronized with slide transitions

To add a presentation, edit `presentations/index.html` and add entries using the template in the HTML comments.

For the audio narration feature, you'll need to:
1. Create a JSON file with slide-by-slide scripts
2. Generate audio using a TTS service (e.g., ElevenLabs, Google TTS)
3. Build a simple JS player that advances slides when each audio segment ends

## Content Migration

Run `python scripts/migrate_content.py` to automatically pull all blog posts and art gallery pages from your current live site and convert them to the new template format. This fetches the HTML content, strips the Hugo template wrapper, and wraps it in the new page template.
